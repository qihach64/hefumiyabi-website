# 瀑布流社交增长功能规划

> 目标：将静态展示层转化为社交互动引擎，提升用户收藏、分享和转化

---

## 核心洞察

和服体验天然具备**社交货币属性**：

1. **视觉冲击力强** — Instagram/小红书爆款内容类型
2. **体验稀缺性** — "我在京都穿和服" = 社交资本
3. **情绪价值高** — 仪式感、文化探索、美的追求

---

## 功能规划

### 1. "Pin to Dream Board" 收藏夹功能

**灵感来源**: Pinterest

**功能描述**:
- 每张图右下角加 ❤️ 收藏按钮
- 用户可创建「我的和服心愿单」
- 未登录用户收藏后引导注册
- 定期邮件提醒用户心愿单中的套餐

**增长飞轮**: 收藏 → 注册 → 邮件提醒 → 转化

**实现状态**: ✅ Phase 1 已实现基础收藏

---

### 2. "拍同款" UGC 入口

**灵感来源**: 抖音/TikTok

**功能描述**:
- 官方图旁边展示真实顾客穿同款的照片
- 标签显示: `#同款打卡 · 326人穿过`
- 点击进入 UGC 瀑布流
- 鼓励用户上传自己的和服照片

**信任飞轮**: 真人图 → 降低决策焦虑 → 转化率↑

**实现状态**: 🔜 P1 待规划

---

### 3. "发现灵感" 智能推荐卡片

**灵感来源**: Instagram Explore

**功能描述**:
在瀑布流中穿插推荐卡片：
- 🌸 「春季热门 · 樱花粉系列」
- 📍 「祇园最出片的5个拍照点」
- 👯 「闺蜜同框 · 双人套餐」
- 基于用户浏览行为个性化推荐

**实现状态**: 🔜 P2 需要数据积累

---

### 4. "一键分享" 社交素材包

**灵感来源**: Canva + Airbnb

**功能描述**:
点击图片后提供：
- 预设文案: "京都和服体验｜这套绝美！"
- 一键生成小红书/Instagram 尺寸图
- 带品牌水印但不突兀
- 支持复制链接、下载图片

**病毒传播**: 用户帮你做免费广告

**实现状态**: ✅ Phase 1 已实现基础分享

---

### 5. "虚拟试穿 → 分享对比图"

**灵感来源**: Snapchat/美图

**功能描述**:
- 官方图旁边加 "试穿这套" 按钮
- AI 生成用户穿这套和服的效果图
- 分享前后对比图到社交媒体
- Hook: "看看我穿这套会是什么样"

**实现状态**: 🔜 P1 已有 AI 基础，需加入口

## 实现阶段

### Phase 1 - MVP (当前)

**目标**: 验证用户对收藏/分享的需求

**功能**:
- [x] 图片 hover 显示 ❤️ 收藏按钮
- [x] 图片 hover 显示 📤 分享按钮
- [x] 收藏状态本地存储 (localStorage)
- [x] 分享支持复制链接
- [x] 分享支持原生分享 API (移动端)

**追踪指标**:
- 收藏点击率
- 分享点击率
- 收藏后注册转化率

**目标**: 将收藏与用户账号绑定

**功能**:

- [ ] 收藏提醒邮件
- [ ] 虚拟试穿按钮接入

### Phase 3 - UGC 生态

**目标**: 建立用户生成内容生态

**功能**:
- [ ] 用户上传和服照片
- [ ] 同款展示功能
- [ ] 照片审核流程
- [ ] UGC 激励机制

### Phase 4 - 智能推荐

**目标**: 个性化内容推荐

**功能**:
- [ ] 用户行为追踪
- [ ] 推荐算法

- [ ] 推荐卡片组件

### 分享功能

```typescript
// 使用 Web Share API (移动端优先)
if (navigator.share) {
  await navigator.share({
    title: planName,
    text: shareText,
    url: shareUrl,
  });
} else {
  // 降级到复制链接
  await navigator.clipboard.writeText(shareUrl);
}
```
